Acerca del Sistema

Sistema de Gestión para Empresa de Aberturas

Este sistema está diseñado para gestionar de manera integral todos los aspectos de una empresa dedicada a la fabricación y venta de aberturas (ventanas y puertas).

Características Principales

  • Gestión de pedidos y presupuestos
  • Control de inventario y materiales
  • Seguimiento de producción
  • Administración de clientes
  • Reportes y estadísticas

Tecnologías Utilizadas

Backend

  • • Django 4.2 LTS
  • • Python 3.12
  • • MySQL 8.0
  • • Redis (Cache)

Frontend

  • • Django Templates
  • • HTMX
  • • Tailwind CSS
  • • DaisyUI
